# 020 — trend-viewer 프론트엔드 폴리시 (dev-frontend 적용)

trend-viewer는 매일 여러 플랫폼의 트렌드를 훑는 **데이터 밀도 높은 조회 도구**다.
마케팅 사이트가 아니라 반복 사용하는 유틸리티이므로, 조용하고 스캔하기 좋은 화면이
목표다. 이 문서는 `dev-frontend` 스킬(aesthetics + anti-slop + vanilla 스택 규칙)을
이 프로젝트에 어떻게 적용하는지 확정하는 폴리시다. P7(프론트 분리)과 안정화 루프의
모든 UI 작업은 이 문서를 기준으로 한다.

업스트림은 이미 다크 톤(`#0f1115` off-black — 순수 검정 아님, 통과)에 카드 그리드
구조로 방향이 잡혀 있다. 포팅은 리디자인이 아니라 **보존 + 교정**이다: 지금 동작하는
레이아웃 문법은 유지하되, anti-slop 위반 항목만 교정한다.

---

## 다이얼 설정

| Dial | 값 | 근거 |
|---|---|---|
| DESIGN_VARIANCE | 4 | 도구형 앱. 그리드 문법 유지, 탭별 밀도 변화로 변주 |
| MOTION_INTENSITY | 3 | 반복 사용 도구 — 호버 리프트, 스켈레톤 정도만. 시네마틱 금지 |
| VISUAL_DENSITY | 7 | 카드 그리드 + 숫자 중심. 콕핏까지는 아니고 밀도 높은 데일리 앱 |

## 디자인 토큰 (확정)

- **베이스**: 업스트림 다크 팔레트 유지 — `--bg:#0f1115`, `--panel:#171a21`,
  `--card:#1c2029`, `--border:#2a2f3a`, `--text:#e8eaf0`, `--muted:#9aa3b2`.
- **액센트 1개 원칙**: `--accent:#ff4d6d`(로즈)만 브랜드 액센트로 유지.
  업스트림의 `--accent2:#4d9fff`(블루)는 브랜드 액센트에서 제외하고
  **포커스/인터랙티브 상태 전용 기능색**으로 강등한다 (`:focus-visible` 링,
  기간 토글 활성 상태). 새 UI에 블루를 장식 목적으로 쓰지 않는다.
- **그림자**: 배경 색조로 틴트 (`rgba(15,17,21,…)` 계열), 제네릭 검정 금지.

## 타이포그래피 (교정 대상 1순위)

- 업스트림 `-apple-system` → **Pretendard Variable** (CJK 규칙상 허용, CDN
  jsdelivr `pretendard-dynamic-subset.css`)로 교체. 빌드 도구 없음 유지.
- 숫자(조회수·좋아요·댓글)는 **`font-variant-numeric: tabular-nums`** 필수 —
  정렬 도구에서 숫자 컬럼이 흔들리면 안 된다. 모노 필요 시 JetBrains Mono.
- 텍스트 크기 고정 px 금지 → `clamp()` 유동 타이포. letter-spacing 0.
- 헤더 `트렌드<span>뷰어</span>` 워드마크는 800 웨이트 유지.

## 레이아웃

- 카드 그리드(`auto-fill, minmax`) 문법 유지 — 반응형 붕괴가 이미 올바름.
- 카드 radius 12px → **8px 이하로 조정** (도구형 UI 규칙).
- `aspect-ratio` 고정(16/9, 9/13)과 `.title` min-height 유지 — 로딩 중
  레이아웃 시프트 방지 규칙에 부합, 건드리지 않는다.
- 탭/칩/기간 토글 = segmented control 문법 유지. 모바일(<768px)에서
  단일 컬럼 붕괴 보장, 터치 타깃 ≥44px 확인.

## 상태 (필수 구현 — 안정화 루프의 핵심)

- **로딩**: "불러오는 중..." 텍스트 → 카드 크기와 동일한 **스켈레톤 로더**
  (원형 스피너 금지). 그리드 시프트 없어야 함.
- **빈 상태**: 계정 0개, 검색 결과 0건일 때 다음 행동을 안내하는 구성된 빈 상태.
- **에러**: 소스별 인라인 에러 (X 수집 실패가 릴스 탭을 오염시키지 않게).
  `window.alert()` 금지. 스레드 폴백은 1급 상태로 유지.
- **택타일**: 카드 `:active`에 `scale(0.98)`, 버튼 호버 배경 시프트.

## 접근성

- 시맨틱 HTML: 탭은 `role="tablist"`/`aria-selected`, 버튼은 `<button>`.
- `:focus-visible` 링 (기능색 블루), 키보드로 탭 전환·카드 열기 가능.
- `prefers-reduced-motion` 지원 필수.
- 이모지 아이콘(🔍/🔄/🤖/📸)은 유지하되 `aria-label` 병기 — 단, 새로 추가하는
  UI는 이모지 대신 인라인 SVG 아이콘 사용.

## 금지 목록 (이 프로젝트 컨텍스트)

- 보라 그라디언트, 네온 글로우, 순수 `#000000`, Inter/Roboto/system-ui 폰트.
- 원형 스피너, `height:100vh`(→ `100dvh`), `top/left` 애니메이션(→ transform).
- 마케팅 카피, 히어로 섹션 — 이 앱의 첫 화면은 콘텐츠 그리드다.

## 성능 가드레일

- 애니메이션은 transform/opacity만. 이미지 `loading="lazy"` (뷰포트 밖 카드).
- 이미지 프록시 경유 썸네일은 실패 시 플레이스홀더 배경(#000) 유지.
